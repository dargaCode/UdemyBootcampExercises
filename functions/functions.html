Functions Exercise

Print out the following answers to the console:

  • isEven():
    • Take a single numeric argument and return true if the number is even, or false otherwise.
      • isEven(4) -> true
      • isEven(21) -> false
      • isEven(68) -> true
      • isEven(333) -> true
  • factorial():
    • Take a single numeric argument and return the factorial of that number.
      • factorial(5) -> 120
      • factorial(2) -> 2
      • factorial(10) -> 3628800
      • factorial(0) -> 1
  • kebabToSnake():
    • Take a single kebab-cased string argument and return the snake_cased version.
      • kebabToSnake("hello-world") -> "hello_world"
      • kebabToSnake("dogs-are-awesome") -> "dogs_are_awesome"
      • kebabToSnake("blah") -> "blah"
